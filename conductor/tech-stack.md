# Backend & Logic
- **Language:** Go 1.21+
- **System Interface:** `github.com/creack/pty` for pseudo-terminal interaction and wrapping interactive Terraform commands.
- **Parsing:** Custom regex-based parser for human-readable Terraform/OpenTofu plan outputs.

# UI & Presentation
- **TUI Framework:** `github.com/charmbracelet/bubbletea` (The Elm Architecture for TUI).
- **Styling & Layout:** `github.com/charmbracelet/lipgloss` for terminal styling and Catppuccin Mocha theme implementation.

# Tooling & Infrastructure
- **Build System:** Go Modules (`go.mod`)
- **Version Control:** Git
